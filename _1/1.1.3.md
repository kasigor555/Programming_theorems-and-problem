# Задачи без массивов

#### **1.1.3.** 
Дано целое число **а** и натуральное (целое неотрицательное) число **n**. Вычислите **a<sup>n</sup>**. Другими словами, необходимо составить программу, при исполнении которой значения переменных **а** и **n** не меняются, а значение некоторой другой переменной (например, b) становится равным **a<sup>n</sup>**. (При этом разрешается использовать и другие переменные.)

> **Решение.** Введём целую переменную **k**, которая меняется от **0** до **n**, причём поддерживается такое свойство: **b = a<sup>k</sup>**).

```javascript
function mathPow(a, n) { // b = (a в степени k)
	
	let k = 0;
	let b = 1;

	while (k != n) {		
		k = k + 1;
		b = b * a;		
	}

	return b;
}

console.log(mathPow(4, 3));

```
Другое решение той же задачи:

```javascript
function mathPow(a, n) { // (a в степени n) = b * (a в степени k)
	
	let k = n;
	let b = 1;
	let c = a;

	while (k != 0) {
		if (k % 2 == 0) {
			k = k / 2;
			c = c * c;
		} else {
			k = k - 1;
			b = b * c;
		}
	}

	return b;
}

console.log(mathPow(3, 3));

```
