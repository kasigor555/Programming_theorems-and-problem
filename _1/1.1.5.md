# Задачи без массивов

#### **1.1.5.** 
Даны натуральные числа а, b. Вычислите произведение a · b, используя в программе лишь операции +, -, =, !=.

>**Решение.**

```javascript

function multiplay(a, b) {
  let k = 0; 
  let c = 0;

  // инвариант: c = a * k

  while (k != b) {
   k = k + 1; // k++;
   c = c + a; // c += a;
  }

  // c = a * k и k = b, следовательно, c = a * b

  return c;
}

console.log(multiplay(3, 9));

```
