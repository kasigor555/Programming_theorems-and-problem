# Задачи без массивов

#### **1.1.7.** 
Дано натуральное (целое неотрицательное) число **а** и целое положительное число **d**. Вычислите частное **q** и остаток **r** при делении **а** на **d**, не используя операций **div "/"** и **mod "%"**.

>**Решение.** Согласно определению, a = q · d + r, 0 <= 6 r < d.

```javascript
function dev(a, d) { // a >= 0; d > 0

  let r = a;
  let q = 0;

  // инвариант: a = q * d + r, 0 <= r

  while (r >= d) {    
    r = r - d; // r >= 0}
    q = q + 1;
  }

  console.log('q = ' + q);
  console.log('r = ' + r);

}

dev(15, 4);
```
