# Задачи без массивов

#### **1.1.6.** 
Даны натуральные числа а и b. Вычислите их сумму а + b. Используйте операторы присваивания лишь вида   
**var1 = var2**,  
**var = num**,  
**var1 = var2 + 1**.

>**Решение.**

```javascript
 // инвариант: c = a + k

 function sum(a, b) {
  let k = 0; 
  let c = 0;

  while (k != b) {
   k ++;
   c = a + k;
  }

  return c;
}

console.log(sum(-3, 9));
```
