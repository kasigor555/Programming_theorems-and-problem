# Задачи без массивов

#### **1.1.9.** 
Последовательность Фибоначчи определяется так: a0 = 0, a1 = 1, ak = ak-1 + ak-2 при k > 2, т.е каждое следующее число равно сумме двух предыдущих. Дано n, вычислите an.

>**Решение.** С помощью рекурсии. Не лучщий вариант, слишком ресурсозатратно при двузначных и более числах

```javascript

function fib(n) {

  return n <= 1 ? n : fib(n - 1) + fib(n - 2);
}

console.log(fib(3)); // 2
console.log(fib(7)); // 13
console.log(fib(77)); // 5527939700884757

```

>**Решение** Итеративный способ

```javascript

function fib(n){
  var a = 0, b = 1, temp;

  for(let i = 0; i < n; i++){
    temp = b;
    b += a;
    a = temp;
  }

  return a;
}

console.log(fib(3)); // 2
console.log(fib(7)); // 13
console.log(fib(77)); // 5527939700884757
```
