# Задачи без массивов

#### **1.1.2.** 
Решите предыдущую задачу, не используя дополнительных переменных (и предполагая, что значениями целых переменных могут
быть произвольные целые числа).

> **Решение.**

```javascript
let a = 13;
let b = 24;

a = a + b;
b = a - b;
a = a - b;

console.log ('a = ' + a, 'b = ' + b); // a = 24 b = 13
```
